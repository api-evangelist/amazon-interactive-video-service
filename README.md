# Amazon Interactive Video Service (amazon-interactive-video-service)
Amazon Interactive Video Service (IVS) is a managed live streaming solution that is quick and easy to set up, and ideal for creating interactive video experiences. It provides SDKs and APIs to build engaging live video experiences with low-latency streaming capabilities.

## APIs

### Amazon IVS API
The Amazon IVS API provides programmatic access to create and manage channels, stream keys, playback key pairs, recording configurations, and chat rooms for low-latency interactive live streaming applications.

**Human URL:** [https://aws.amazon.com/ivs/](https://aws.amazon.com/ivs/)
